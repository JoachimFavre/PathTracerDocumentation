Detailed Description

Stores objects groups and a camera for the render.

Constructor & Destructor Documentation

◆ Scene() [1/3]

Scene::Scene ( )

Default constructor.

By default, samplePerPixel is set to 8, minBounces is set to 5, maxDepth is set to 10 and rrStopProbability is set to 0.1.

◆ Scene() [2/3]

Scene::Scene ( PerspectiveCamera  camera,
unsigned int  samplePerPixel,
unsigned int  minBounces,
double  maxDepth,
double  rrStopProbability 
)

Main constructor.

Parameters
cameraThe camera that will be used for the render.
samplePerPixelThe number of ray casted per pixel. Increasing it will give a higher quality, but at a cost of a linearly increasing time.
minBouncesThe minimum number of bounces that each ray will do. It can be less if the ray does not intersect with any object.
maxDepthThe maximum distance each ray can do between two bounces.
rrStopProbabilityThe probability that the ray stops. See russian roulette path termination in my TM's report for further information.
